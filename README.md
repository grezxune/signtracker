Track your child's American Sign Language (ASL) learning journey!

## Features

- **Track Known Signs** - Keep a record of all the ASL signs your child has learned
- **Look Up New Signs** - Browse 100+ common baby/toddler ASL signs with descriptions and video links
- **Family Sharing** - Share your child's progress with your spouse or family members
- **Progress Stats** - See statistics on mastered, familiar, and learning signs
- **Multiple Children** - Track progress for multiple children

## Tech Stack

- **Next.js 16** - React framework
- **Tailwind CSS** - Styling
- **Convex** - Backend database & real-time sync
- **@convex-dev/auth** - Authentication

## Getting Started

1. Clone the repo
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up Convex:
   ```bash
   npx convex dev
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```

## Environment Variables

Copy `.env.example` to `.env.local` and fill in your values.
